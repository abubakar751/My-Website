# Abu Bakar — Portfolio

Premium personal portfolio. **Plain React + Vite + JavaScript (JSX)**.

## Stack
- React 18 + Vite 5
- React Router DOM 6
- Tailwind CSS 3
- Framer Motion 11
- React Helmet Async

## Run locally
```bash
npm install
npm run dev
```

## Build for production
```bash
npm run build
```
Output: `dist/`

## Deploy to Netlify
This repo includes `netlify.toml` and `public/_redirects`. Connect the repo to
Netlify and it deploys as-is. Build command `npm run build`, publish dir `dist`.

## Edit contact info
Update placeholder phone/email/links in `src/data/contact.js`.
